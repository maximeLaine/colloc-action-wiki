# Session 2 — Le Comte de Valombre

**Lieu :** Château de Valombre (grande salle elfique, bannières vert & argent)
**PNJ rencontrés :** Comte Aldren de Valombre, Gareth (☠ mort cette session)

---

## Résumé

Audience avec le **Comte Aldren**, haut-elfe hautain qui accuse ouvertement Dornlac. Les PJ flairent qu'il cache quelque chose — une rançon a été reçue. Des indices dans le château pointent vers un intermédiaire lié à **Malefosse**.

## Événements clés

1. **Audience** — Aldren accuse Dornlac avec véhémence, mais un jet d'Intuition révèle qu'il dissimule la vérité
2. **Indices au château** — plume noire huilée + poudre camphrée (apothicaire lié à Malefosse)
3. **Rencontre avec l'espion** — les bandits tentent d'éliminer les PJ, puis fuient (billets codés brûlés)
4. **Mort dans la nuit** — Gareth est assassiné. Combat contre une **Guenaude verte**

## Cliffhanger

> Le Comte de Valombre envoie l'équipe négocier avec le Comte de Dornlac.

## Indices découverts

- Poudre camphrée → apothicaire **Feu-froid** lié à Malefosse
- Rançon dissimulée par Aldren → Élia vaut plus qu'il ne le dit
- Billets codés brûlés → réseau d'espions actif
